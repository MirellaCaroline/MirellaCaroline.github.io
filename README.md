# MirellaCaroline.github.io

<b>link do jogo:</b> <https://resplendent-chebakia-d8b273.netlify.app/>

<b>Sobre o jogo:</b> é um pequeno jogo de plataforma, onde Estrelar (personagem) precisa salvar a Silkie (animal de estimação da Estrelar).

<b>Objetivo:</b> Estrelar precisa coletar todos os corações e coletar a chave (após derrotar o Robin) para salvar a Silkie .

<b>Comandos:</b> arrow key (movimento do personagem), tecla S (poder do personagem), tecla M (reinicia o jogo).
